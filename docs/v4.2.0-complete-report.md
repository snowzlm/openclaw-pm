## ✅ 完成功能

### 未回复消息检查器 (UnansweredChecker)

**核心功能**:
- 扫描所有 agent 的 session 文件
- 检测最后一条消息是否为 `user` 角色
- 支持按时间过滤（默认 24 小时内活跃）
- 支持按 agent 过滤
- 自动恢复功能（发送恢复消息或 wake 通知）

**实现细节**:
```typescript
// 检查逻辑
1. 遍历 ~/.openclaw/agents/*/sessions/*.jsonl
2. 读取每个文件的最后一行
3. 解析 JSON，检查 message.role 或 role 字段
4. 如果是 "user" 则标记为未回复
5. 提取 sessionKey、timestamp、preview 等信息
```

**自动恢复机制**:
```typescript
// 优先级
1. 尝试使用 openclaw sessions send 发送恢复消息
2. 失败则回退到 openclaw cron wake 发送通知
3. 记录成功/失败数量
```

### 1. src/unanswered-checker.ts (6133 字节)
**核心类**: `UnansweredChecker`

**方法**:
- `check(options)`: 主检查方法
- `checkSession(path, agentId, options)`: 检查单个 session
- `recoverSessions(sessions)`: 批量恢复
- `sendWakeNotification(text, mode)`: 发送 wake 通知

**接口**:
```typescript
interface UnansweredSession {
  sessionKey: string;
  agentId: string;
  sessionName: string;
  timestamp: string;
  preview: string;
  lastModified: Date;
}

interface UnansweredCheckResult {
  unanswered: UnansweredSession[];
  count: number;
  recovered?: number;
  failed?: number;
}
```

## 🎯 CLI 使用示例

```bash
未回复会话数: 0

✓ 没有发现未回复的消息
```

### 包括旧 session
```bash
$ openclaw-pm check-unanswered --all
```

```bash
$ openclaw-pm check-unanswered --agent main
```
