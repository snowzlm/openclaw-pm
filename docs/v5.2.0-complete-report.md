## 技术细节

### Logger 测试策略
1. **文件日志测试**: 使用临时目录和文件，测试后自动清理
2. **级别过滤测试**: 验证不同日志级别的过滤逻辑
3. **格式验证**: 使用正则表达式验证时间戳格式
4. **Mock 控制台**: 使用 Jest spy 验证控制台输出
5. **边界情况**: 测试空消息、多行消息、特殊字符

### 修复的问题
1. **enableFile 配置**: Logger 只在 `enableFile=true` 时创建目录和写入文件
2. **重复属性**: 修复 sed 命令导致的 `enableFile` 重复定义
3. **测试隔离**: 每个测试使用独立的临时文件，避免相互影响
